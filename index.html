Calculating the nth binary digit of pi

Calculating the nth binary digit of pi is possible without having to know the previous digits, amazingly enough, and there is a wonderful formula for it! The original papers can be found at Peter Borwein's site. I used material from On the Rapid Computation of Various Polylogarithmic Constants, by David Bailey, Peter Borwein and Simon Plouffe to write my program. Download it in PostScript or PDF format.

You can find the binary expansion (32768 digits) on this web page. The same data is in a text file for easy processing. You can also get an up-to-date check on the status of nth binary digit calculation records. There's even a distributed effort to calculate binary digits of pi.

My program is a Java implementation of Bailey, Borwein and Plouffe's algorithm to calculate the nth binary digit of pi. Currently the code runs only as command line applications. You can browse the JavaDoc online, or download the code.
